# Web-Based Lexical Analyzer

A web application that converts source code into lexemes and tokens with maximum accuracy. Built as a university project for Formal Language & Automata Theory.

## Features

-  **Hand-written State Machine** – 100% accurate tokenization
-  **Multi-language Support** – C, C++, Python, Java, and more
-  **File Upload** – Supports `.txt`, `.py`, `.cpp`, `.docx`, `.java`
-  **Token Table** – Displays Lexeme, Token Type, Line, Column
-  **Error Handling** – Never crashes on invalid input
-  **Modern UI** – Clean, responsive, auto-grow textarea

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Python 3.10+, Flask
- **Tokenization:** Hand-written Deterministic Finite Automaton (DFA)
